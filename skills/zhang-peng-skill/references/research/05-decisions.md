# 张鹏（极客公园创始人）：重大决策与职业历程

> 调研日期：2026-04-08
> 信息源：新浪科技、36kr、量子位、LinkedIn、世界经济论坛、chinadaily、pitchhub、jili20.com

## 一、职业轨迹：从记者到创始人

### 1998–2009：媒体人阶段

张鹏大学毕业后（约1998年）进入媒体行业，最初在《IT经理世界》做记者。因资历尚浅被分配观察互联网行业——彼时腾讯、Google刚刚成立，互联网属于冷门领域。这一"偶然"分配，奠定了他15年媒体生涯的核心主轴。

- 从普通记者升至《IT经理世界》执行总编
- 2009年：作为联合创始人创办《商业价值》杂志，任主编
- 编辑部出产的"摆渡人"效应：知乎创始人周源、摩拜创始人胡玮炜、网易新闻客户端首任内容负责人吴茂林均出自其麾下

### 2010–2013：双线并行期

- 2010年：以"个人爱好"形式创办极客公园，每月在朝阳公园地下室组织两次四五十人的产品经理交流
- 2013年：主持谷歌董事长埃里克·施密特对谈；年底卸任《商业价值》主编，全身心投入极客公园
- 关键节点：雷军当时做小米还没人信，微信还没人用，张一鸣还坐在观众席里

### 2014至今：创始人阶段

- 2014年：极客公园正式注册独立公司，接受机构投资（小米/顺为联合投资B轮数千万元）
- 2017年：成立科技企业家社群「前沿社」、旗下早期基金「变量资本」
- 2019年起：受邀加入世界经济论坛（WEF）社区，列于官方人物目录
- 近年：孵化创业者社区「Founder Park」，聚焦AGI时代

## 二、极客公园的创立背景与初心（2010年前后）

### 决策起点

极客公园的诞生**不是商业决策，而是情感判断**。张鹏在12年媒体工作中发现，互联网领域的从业者有别于其他行业——他们有创新的想法，有极强的行动力。他的直觉是：把这些人聚在一起，将激发远超单个个体的创新价值。

最初形态极为简单：每月两次，地下室，四五十人，自由交流。没有商业模式，没有公司结构，纯粹是兴趣驱动。

### 时间窗口的重要性

2010年前后，正是中国移动互联网爆发的前夜：
- 微信尚未诞生
- 小米尚未获得市场认可
- 今日头条尚未创立
- 张一鸣、王兴、雷军都还在摸索期

这批人在极客公园的地下室相遇，构成了中国互联网史上一个隐秘但极为重要的早期网络节点。

## 三、从纯媒体到「创新社区」的转型决策

### 转型逻辑

张鹏没有将极客公园做成另一个垂直科技媒体（如36氪、虎嗅、钛媒体），而是选择了一条差异化路径：**以社区和活动为载体，做创新者之间的连接器**。

这一选择的核心逻辑是：

| 维度 | 传统资讯媒体 | 极客公园选择的路径 |
|------|------------|-----------------|
| 目标函数 | 流量、覆盖面 | 激发「变量」、点亮非共识创业者 |
| 核心资产 | 内容版权 | 人脉网络与社区信任 |
| 随时间变化 | 内容老化快 | 社区价值随时间积累增厚 |
| 商业护城河 | 低，易被复制 | 高，关系网络不可复制 |

张鹏的原话概括了这一逻辑："如果过去媒体好内容的目标是流量，那么极客公园作为一个创业者社区，无论日常的内容还是年度的大会，它的目标就应该是激发变量——对于科技趋势，对新一代优秀产品和创业者这些变量的识别和点亮。"

### 自我认知的升级

张鹏坦承，当他从旁观的媒体人变为自己也管理一家公司时，他对企业家思维和心境有了全新的"同理心"——这是他此前作为记者难以具备的视角。这种同理心反过来强化了他作为"连接者"而非"报道者"的角色定位。

## 四、极客公园创新大会（GIF/IF）的创办与演化

### 起源

- 2010年9月：举办首届"中国互联网创新产品评选"
- 2010年10月20日：第一期线下活动"百度开放平台带来的新机会"
- 2011年：正式命名为极客公园创新大会；同年微信团队首次登台，雷军首次公开讲述小米战略
- 2012年1月：正式举办"第一届中国互联网创新大会"

### 历届重要节点

| 年份 | 重要事件 |
|------|---------|
| 2012 | 正式命名为极客公园创新大会 |
| 2013 | 第三届，谷歌董事长埃里克·施密特受邀演讲 |
| 2014 | 第四届，特斯拉马斯克首次在华公开亮相，同台的还有张一鸣 |
| 2015 | 场地扩展至北京751D·PARK |
| 2016 | 主题"DARE TO CREATE" |
| 2017 | 主题"Hello, Neo Generation!"；参与SXSW Austin的"China Gathering" |
| 2019 | 十周年纪念大会 |
| 2021 | 首次落地深圳 |
| 2023 | GIF 2024，主题"增强人生"，聚焦AI大模型创业机遇 |
| 2025 | GIF 2026，主题"进程由我（On The Loop!）"，在北京798艺术区举办，已第16届 |

### 大会的差异化定位

GIF大会不同于一般行业会议的关键在于：**不是聚集今天最明星的人，而是找到正在创造未来变量的人**。这一选择标准使得大会具有前瞻性，也解释了为何它能持续吸引真正的先锋人物，而非仅仅是当红明星。

国际嘉宾包括：埃隆·马斯克（Tesla）、埃里克·施密特（Google）、史蒂夫·沃兹尼亚克（Apple）、特拉维斯·卡兰尼克（Uber）。

## 五、商业模式选择：为何是活动+社区，而非资讯媒体

### 核心选择

极客公园最终形成了三层商业结构：

1. **内容媒体层**：观察前沿科技，产出有深度的报道和分析，构建品牌公信力
2. **活动/会展层**：以年度GIF大会为旗舰，辅以专题活动、工作坊，产生直接收入
3. **社区/投资层**：前沿社（企业家社群）+ 变量资本（早期基金）+ Founder Park（创业者社区）

### 商业模式的优越性

这一结构比纯媒体路线有三重优势：
- **稀缺性更强**：内容可被复制，但张鹏与雷军、张一鸣等人的十年信任关系不可复制
- **收入多元化**：活动赞助、会员服务、投资回报，不依赖单一广告收入
- **正向飞轮**：社区吸引更优秀的人，优秀的人使社区更有价值，价值反过来吸引更多人加入

### 融资验证

- B轮融资由小米/顺为联合投资（数千万人民币），顺为系支持本身就是雷军对张鹏判断力的背书

## 六、张鹏在创业者生态中的角色定位

### "摆渡人"而非"意见领袖"

互联网评论人keso（何楠）在2018年一篇文章中，将张鹏定义为中国科技圈的"摆渡人"。这一定义捕捉了一个微妙的区别：张鹏不是以输出"正确观点"建立影响力的思想领袖，而是通过**让对的人相遇、让信息流向正确位置**来发挥作用。

### 具体连接行为证据

- 张小龙做微信后的**第一个杂志封面报道**由张鹏操刀
- 王兴做美团后的**第一个封面报道**由张鹏操刀
- 雷军做小米后的**第一个杂志封面报道**由张鹏操刀
- 2014年，张一鸣尚未"一鸣惊人"时，张鹏安排他参加极客公园硅谷行，与马斯克、苹果前CEO等人会面；张一鸣那次行程后写的《中国科技公司的"黄金时代"》，被认为是后来TikTok全球化的思考起点
- 2014年，将马斯克和张一鸣同台放在GIF大会，明确表示两者都是"非共识"的押注对象

### 人格特质的护城河

区别于其他媒体人的关键特质：
- **不功利**：知道的比谁都多，但极少爆料自我刷存在
- **早期押注**：发现大疆、今日头条时，外界还未认可
- **同理心**：自己也是创业者，真正理解企业家的处境

## 七、极客公园与头部科技公司的关系建立逻辑

### 关系网络的形成机制

张鹏与头部科技公司的关系，并非通过"采访"建立，而是源于**共同经历的早期不确定性时期**。他在这些创始人还未成名时就给予了平台、背书和媒体关注，形成了基于互信的长期关系。

这解释了一个现象：为何这些CEO愿意年复一年出现在极客公园的舞台上，接受张鹏的深度对谈——因为他们彼此有跨越多年的历史积累，不是临时的商务关系。

### 国际关系网络

- 参与世界经济论坛（WEF），列于官方人物目录
- 2017年率队参加奥斯汀SXSW，举办"China Gathering"，将中国科技创新故事带向全球
- 历届GIF大会邀请全球科技领袖来华，将极客公园打造为中西方科技人士的交汇节点

## 八、近年极客公园在AI时代的转型尝试

### 内容层面的转型

从报道移动互联网转向全面覆盖AI，极客公园成为中国AI创业圈的重要内容平台。2023年起，GIF大会主题全面向AI时代转型：
- IF 2024主题："增强人生"——关注大模型如何增强人类能力
- IF 2026主题："进程由我（On The Loop!）"——从被动接受AI到主动塑造AI时代

### 组织层面的转型

孵化「Founder Park」社区，专注AGI时代创业者：
- 通过"产品市集"帮助早期产品获得真实用户反馈
- 通过"出海增长研坊"帮助超过340位创业者构建全球化认知

### 资本层面的转型

「变量资本」当前聚焦AI创新、硬件创新等领域：
- 已投项目包括小鹏汽车、宇树科技、连咖啡等10余个项目
- 管理四只早期基金
- 秉持"Founders backing founders"理念，由已成功的创业者投资新一代创业者

### 张鹏对AI时代的判断框架

他用"节气"比喻AI创业的阶段：
- ChatGPT出现 = 立春
- 2024年 = 雨水（探索期）
- 2025年 = 惊蛰（应用开始冒芽）
- 2026年 = 春分（全面开花）

这一判断框架反映了他一贯的方法论：不追热点，而是判断周期位置，在"非共识"阶段押注。

## 核心决策模式总结

| 决策 | 时间 | 选择 | 深层逻辑 |
|------|------|------|---------|
| 创立极客公园 | 2010 | 以个人爱好形式，非商业驱动 | 先建社群信任，再谈商业变现 |
| 全职投入 | 2013 | 卸任《商业价值》主编 | 到了必须all-in的节点 |
| 商业模式选择 | 2014 | 活动+社区，拒绝纯媒体路线 | 关系网络比内容更难被复制 |
| 前沿社+变量资本 | 2017 | 从服务创业者到与创业者同行 | 深度参与比旁观有更大价值 |
| AI时代转型 | 2023+ | 内容+社区+资本全面转向AI | 延续"发现变量"的核心使命 |
