# Sam Altman · 完整时间线

> 调研日期：2026-04-08
> 覆盖范围：1985年出生 → 2026年4月

---

## 早年与教育（1985-2005）

| 时间 | 事件 |
|------|------|
| 1985-04-22 | Samuel Harris Altman 出生于芝加哥伊利诺伊州，随后随家人迁居密苏里州圣路易斯郊区。父亲为房地产经纪人，母亲 Connie Gibstine 为皮肤科医生，家中排行长子，共四兄妹。 |
| ~1993（约8岁） | 收到人生第一台电脑——苹果 Macintosh——开始自学编程、拆解硬件。12岁时开始在网上倒卖棒球卡，完成第一次"创业实验"。 |
| ~1998-2003 | 就读 John Burroughs School（圣路易斯私立精英中学）。 |
| 2003 | 进入斯坦福大学计算机科学专业。后回忆称，从教授讲课学到的东西远不如和同学打扑克学到的多——扑克教会他"在极不完整信息下判断模式、做决策"。 |
| 2005 | 就读斯坦福两年后辍学，年仅19岁，联合创立 Loopt。 |

## Loopt 时期（2005-2012）

| 时间 | 事件 |
|------|------|
| 2005 | 与 Nick Sivo、Alok Deshpande 共同创立 Loopt（前身名 Radiate），核心产品：基于 GPS 的实时位置共享移动应用。同年入选 Y Combinator **第一批**孵化企业（与 Reddit 同期），获 YC 种子资金 6,000 美元。 |
| 2005-2007 | 获得 New Enterprise Associates（Patrick Chung 主导）500 万美元 A 轮，随后 Sequoia Capital 跟投，总融资超过 3,000 万美元。 |
| 2007-2011 | Loopt 达到估值峰值约 1.75 亿美元，但始终未能突破用户增长瓶颈。Altman 反思：产品在智能手机普及之前，底层平台（运营商）太受限制，时机略早。 |
| 2012-03 | Loopt 被银行公司 Green Dot 以 **4,340 万美元**现金收购，七年创业画上句点——财务上属于"正向退出"但远未达预期。 |

## YC 时期（2012-2019）

| 时间 | 事件 |
|------|------|
| 2011 | Paul Graham 邀请 Altman 以 **合伙人（Partner）** 身份加入 YC。Altman 通过了 PG 所谓的"年轻创始人测试"：能否有效领导比自己年长的成年人。 |
| 2014-02 | 接替 Paul Graham，正式成为 **Y Combinator 总裁（President）**。任期内监管 Airbnb、Dropbox、Stripe 等明星公司的成长。 |
| 2015-12 | 与 Elon Musk、Greg Brockman、Ilya Sutskever、Wojciech Zaremba 等人共同创立 **OpenAI**，以非营利机构形式注册，获 10 亿美元初始承诺捐赠（包括 Musk 和 Thiel 等人）。Altman 担任董事会主席，但最初日常运营由 Sam Altman 与 Greg Brockman、Ilya Sutskever 共同负责。 |
| 2016 | Altman 成为 **YC Group 总裁**，统管 Y Combinator 本体、YC Continuity（成长期基金）、YC Research（非营利研究院，后演变为 OpenResearch）。 |
| 2018 | Elon Musk 以"与 Tesla AI 研究存在利益冲突"为由，**辞去 OpenAI 董事会**席位（此后双方矛盾逐步公开化）。 |
| 2019 | 从 YC 离职（担任总裁约 5 年），全身心转向 OpenAI。同年 OpenAI 由纯非营利转型为 **"有上限利润"（Capped-profit）** 混合结构，吸引外部投资。 |

### 2023年11月 Board Crisis（逐日）

#### 背景
OpenAI 内部长期存在两种文化：以 Ilya Sutskever 为代表的"安全优先/非营利使命派"，与以 Altman 为代表的"快速商业化/消费产品派"。2023 年 10 月，Altman 削减了 Sutskever 的职权，双方裂痕加深。与此同时，Altman 正密谋耗资数千亿美元的 AI 芯片计划及与软银孙正义的合作，部分董事认为此举绕过了董事会授权。

---

| 日期 | 事件 |
|------|------|
| **11月16日（周四）晚** | Altman 收到 OpenAI 联合创始人发来的短信，要求他次日参加 Google Meet 视频通话。此时，部分董事会成员已秘密与 CTO Mira Murati 接触，属意她接任 CEO。 |
| **11月17日（周五）上午** | 董事会紧急会议召开。四位现任董事（Ilya Sutskever、Helen Toner、Tasha McCauley、Adam D'Angelo）以**5:4 多数**（含 Altman 自己反对）决定解雇 Altman。官方声明称："董事会对他继续领导 OpenAI 的能力失去信心……他与董事会沟通时并不一贯坦诚，妨碍了董事会履行职责。" |
| **11月17日（周五）中午** | 微软在公告发出**"一分钟前"**才获通知。Greg Brockman 被告知免去董事会主席职务（但保留总裁职位），Brockman 当日下午宣布辞职。三名资深研究员 Jakob Pachocki、Aleksander Madry、Simon Sidor 随即宣布离职。Mira Murati 被任命为**代理 CEO**。 |
| **11月18日（周六）** | 微软发表声明支持 Murati 领导班子。董事会私下开始向 Altman 试探"能否回来"，Altman 据报拒绝接受包含追责条款的方案。 |
| **11月19日（周日）** | 董事会宣布任命前 Twitch CEO **Emmett Shear** 为新代理 CEO，绕过 Murati，引发公司内部爆发——员工"从办公室流出，互相拥抱"。当晚，微软 CEO **Satya Nadella 宣布**将聘用 Altman 领导微软新 AI 部门，Brockman 等人随行加入。Altman 在 X 上发推："使命继续。" |
| **11月20日（周一）** | 接近 800 名 OpenAI 员工（约 95%）联署公开信，要求董事会辞职并恢复 Altman 职务，威胁集体辞职转投微软。联署人数之多一度让 Google Docs 崩溃。首席科学家 Ilya Sutskever 公开发推："我为参与此次行动深感懊悔。"员工在旧金山湾区各地聚集，取消感恩节计划研究对策。 |
| **11月21日（周二）** | Mira Murati 在 X 发布她与 Altman、Brockman 等人在 Altman 家中聚会的合影，配文"still working on it…"。当晚 10 点后，OpenAI 宣布已达成"原则性协议"，Altman 将回归担任 CEO，新董事会主席将由前 Salesforce 联席 CEO **Bret Taylor** 出任。 |
| **11月22日（周三）** | **Altman 正式复职**。新初始董事会成员：Bret Taylor（主席）、Larry Summers、Adam D'Angelo（唯一留任原董事）。微软获一席**无投票权的董事会观察员**席位。 |
| **11月29日（周三）** | 官方文件正式确认 Altman 复任 CEO，Brockman 复任总裁。Altman 对 Axios 表示，新董事会的"最优先任务"是探索治理结构改革，包括审视 OpenAI 是否应继续由非营利机构管控。 |

#### 事后调查与延伸影响
- WilmerHale 律所事后调查结论：Altman 的行为"未达到强制撤职的程度"。
- 2024年5月，前董事 Helen Toner 接受采访：Altman 曾隐瞒 ChatGPT 发布消息、对 OpenAI 创业基金的持股，以及对公司安全流程的虚假陈述；另有两名高管向董事会提交了 Altman 对其"心理施压"的书面证据。
- 2026年4月，《纽约客》Ronan Farrow 发表长达 18 个月调查结果，引用逾 100 名受访者和 200 余页内部文件，进一步坐实上述指控（见"2026年最新"章节）。
